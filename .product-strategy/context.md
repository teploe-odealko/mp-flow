# Project Context

## What We Build
MPFlow -- открытая AI Agent First система управления продажами на маркетплейсах (Ozon FBO).
Ядро: каталог, FIFO складской учет, закупки, продажи, финансы (P&L), аналитика (unit economics), управление ценами, MCP AI-агент (54+ инструментов).
Архитектура плагинов: definePlugin() система расширений.

## Core Technologies
- Server: Hono 4 + MikroORM 6 + awilix DI + iron-session
- Client: React 19 + Vite 6 + Tailwind CSS + shadcn/ui
- Plugins: Ozon (маркетплейс интеграция), Ali1688 (привязка товаров к 1688.com)
- MCP: 54+ инструментов для AI-агентов (Claude, ChatGPT, Manus)

## Current Stage
Early Growth -- продукт в проде, есть лендинг, облачная версия на admin.mp-flow.ru

## Positioning
"Open source ERP для маркетплейсов. FIFO учет, юнит-экономика, управление ценами на Ozon. AI Agent Native."
