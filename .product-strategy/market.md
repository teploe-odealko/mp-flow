- Селлеры на Ozon: 500k+ активных продавцов
- Селлеры на WB: 700k+ активных продавцов
- TAM (инструменты для селлеров): ~1.2M продавцов, ~500-5000 руб/мес = ~7-72 млрд руб/год
- SAM (Ozon FBO, малый/средний бизнес, нужен учет): ~100-200k продавцов
- SOM (реалистично для open source нишевого продукта): 500-2000 пользователей в первый год

## Key Differentiators of MPFlow
1. Open source (единственный в нише)
2. AI Agent Native (54+ MCP tools -- уникально на рынке)
3. FIFO учет по партиям (профессиональный уровень)
4. Self-hosted вариант (для тех кто боится за данные)

## Trends
- Банки (Тинькофф, Точка) дают бесплатную базовую аналитику -- давление на low-end
- AI-агенты и MCP -- растущий тренд, еще не освоен конкурентами
- Селлеры хотят "все в одном" -- не 5 разных сервисов
- Рынок moving from "аналитика конкурентов" к "управление своим бизнесом"

## Price Sensitivity
- Начинающие селлеры: 0-500 руб/мес (очень чувствительны к цене)
- Растущие (100-500 заказов/мес): 500-2000 руб/мес
- Зрелые (500+ заказов/мес): 2000-5000 руб/мес, готовы платить за экономию времени
- Крупные: 5000-20000 руб/мес, ценят автоматизацию
