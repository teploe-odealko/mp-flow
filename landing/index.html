MCP Protocol

AI-агент, который реально управляет магазином

Не чат-бот с советами, а полноценный агент с доступом к каталогу, складу, ценам, закупкам, аналитике и всему Ozon API. Скажите словами — он сделает.

Работает через MCP (Model Context Protocol) — открытый стандарт. Подключается к любому AI-клиенту за 2 минуты.

Частые вопросы

MPFlow — это open source ERP с AI-агентом (MCP), а не просто аналитический дашборд. Ключевые отличия: учёт партий с точной себестоимостью, управленческие отчёты PnL и ДДС, система плагинов, self-host бесплатно навсегда. Код открыт — можно дорабатывать под себя.

Сейчас полностью поддерживается Ozon FBO: синхронизация продаж, финансов, остатков, возвратов. Архитектура плагинов позволяет добавить поддержку любого маркетплейса.

Создайте API-ключ в Настройках, добавьте конфиг .mcp.json в ваш проект с URL и ключом. 2 минуты — и агент готов к работе. Подробная инструкция — в документации.

Подвоха нет. Self-host: docker compose up -d — и все функции без ограничений. Бесплатно навсегда. Мы зарабатываем на облачной версии (300 ₽/мес) для тех, кому не хочется разворачивать сервер.

При self-host все данные на вашем сервере — мы их даже не видим. В облаке: шифрованные сессии, API-ключи хешируются. Код открыт — можете проверить сами, что делает система с вашими данными.
