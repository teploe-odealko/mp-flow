Продажи FBO

Частые вопросы

MPFlow — это open source ERP с нативной поддержкой AI-агентов (MCP), а не просто аналитический сервис. Ключевые отличия: учёт партий с себестоимостью, управленческие отчёты PnL и ДДС, система плагинов, возможность self-host бесплатно. Код открыт — можно дорабатывать под себя.

Сейчас полностью поддерживается Ozon FBO: синхронизация продаж, финансов, остатков, возвратов. Архитектура плагинов позволяет добавить поддержку любого маркетплейса.

Создайте API-ключ в Настройках, добавьте конфиг .mcp.json в ваш проект с URL и ключом. Подробная инструкция — в документации.

Да. docker compose up -d — и система работает. Все функции доступны без ограничений. Нужен сервер с Docker (VPS от 1 GB RAM).

При self-host все данные на вашем сервере. В облаке: шифрованные сессии, API-ключи хешируются. Код открыт — можете проверить сами.
