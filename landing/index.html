Складской учёт по партиям — точная себестоимость каждой единицы

Каждая закупка создаёт отдельный лот с полной разбивкой затрат. При продаже система автоматически списывает самые старые партии (FIFO) и рассчитывает реальную себестоимость проданного товара.

Калькулятор прибыли Ozon — установите цену с нужной маржой

Система знает все комиссии Ozon по категориям, логистику, хранение, возвраты и налоги. Поставьте цену с нужным ROI одной кнопкой — или задайте целевую маржу, и система рассчитает цену сама.

  • Все комиссии Ozon по категориям
  • Forward: цена → маржа
  • Reverse: маржа → цена
  • Массовое обновление цен

Частые вопросы

MP Flow — это open source ERP, а не просто аналитический сервис. Ключевые отличия: FIFO учёт по партиям с точной себестоимостью, полноценный MCP сервер для AI-агентов (54+ инструментов), возможность self-host на своём сервере бесплатно. Код открыт — вы можете дорабатывать систему под себя.

Сейчас полностью поддерживается Ozon FBO: синхронизация продаж, финансов, остатков, возвратов, поставок. Система проектировалась как мульти-маркетплейсная — добавление Wildberries и других площадок в планах.

Да. docker compose up -d — и система работает. Все функции доступны без ограничений. Нужен сервер с Docker (VPS от 1 GB RAM).

Создайте API ключ в настройках, укажите URL MCP сервера в вашем AI-клиенте (Claude Code, ChatGPT и др.). Подробная инструкция — в документации.

При self-host все данные хранятся на вашем сервере. В облаке: Ozon API ключи шифруются через pgcrypto, доступ по JWT токенам. Код открыт — можете проверить сами.

Да. 84 REST API endpoints с документацией на proxy.mp-flow.ru/docs + 54 MCP инструментов для AI-агентов. Пользовательская документация на docs.mp-flow.ru.
