## AI Agent Native

MPFlow спроектирован с поддержкой AI-агентов с первого дня. Через [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) любой AI-клиент получает доступ ко всем функциям системы.

## Что может AI-агент

С помощью 54 MCP инструментов агент может:

- **Управлять каталогом** — искать товары, создавать карточки, обновлять данные
- **Вести закупки** — создавать заказы поставщикам, проводить приёмку
- **Анализировать продажи** — строить отчёты юнит-экономики, PnL, ДДС
- **Управлять ценами** — рассчитывать оптимальные цены с учётом всех комиссий
- **Синхронизировать с Ozon** — импортировать товары, остатки, продажи, финансы
- **Планировать закупки** — генерировать планы на основе истории продаж
- **Контролировать логистику** — отслеживать перемещения и поставки

## Пример использования

После подключения агента (см. [настройка MCP](/docs/ai-agents/mcp-setup)) можно давать команды на естественном языке:

> «Покажи юнит-экономику за последний месяц. Какие товары убыточны?»

> «Установи цены на все товары с ROI не менее 40%»

> «Синхронизируй продажи за эту неделю и покажи PnL»

> «Сгенерируй план закупок на следующий месяц»
