## Обзор

OpenMPFlow предоставляет **54 MCP инструмента** по 13 доменам. Все инструменты доступны через MCP протокол любому подключённому AI-клиенту.

## Каталог (6 инструментов)

| Инструмент | Описание |
|------------|----------|
| `catalog_list` | Список товаров с поиском и пагинацией |
| `catalog_get` | Детали товара по ID |
| `catalog_create` | Создание карточки товара |
| `catalog_update` | Обновление полей товара |
| `catalog_preview_1688` | Предпросмотр товара с 1688.com |
| `catalog_import_1688` | Импорт данных поставщика с 1688.com |
