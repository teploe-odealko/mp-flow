## Требования

- [Docker](https://docs.docker.com/get-docker/) с Docker Compose v2
- Git

## Установка

```bash
git clone https://github.com/nickthecook/mpflow
cd mpflow
cp .env.example .env
docker compose up
```

Запустятся три сервиса:

| Сервис | URL | Описание |
|--------|-----|----------|
| Admin UI | [localhost:3000](http://localhost:3000) | Веб-интерфейс |
| Proxy API | [localhost:8000](http://localhost:8000) | Backend API |
| PostgreSQL | localhost:5432 | База данных |

## Первый вход

Откройте [http://localhost:3000](http://localhost:3000). Войдите с:

- **Логин:** `admin`
- **Пароль:** `admin`

<Callout type="warn">
Обязательно смените пароль в продакшене! Задайте `ADMIN_BOOTSTRAP_PASSWORD` в `.env`.
</Callout>

## Подключение Ozon

1. Перейдите в раздел **Каталог**
2. Нажмите кнопку **Ozon** (вверху справа)
3. Введите Ozon Client-ID и API-Key
4. Нажмите **Сохранить креды**

API-ключи можно получить на [seller.ozon.ru/app/settings/api-keys](https://seller.ozon.ru/app/settings/api-keys).
