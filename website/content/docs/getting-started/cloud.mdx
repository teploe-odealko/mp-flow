## Облако mp-flow.ru

Облачная версия — самый быстрый способ начать работу с OpenMPFlow. Не нужно ничего устанавливать.

### Что входит

- Все функции OpenMPFlow без ограничений
- Автоматические обновления
- Ежедневные бэкапы
- Приоритетная поддержка
- HTTPS и шифрование данных

### Начало работы

1. Перейдите на [admin.mp-flow.ru](https://admin.mp-flow.ru)
2. Создайте аккаунт
3. Подключите Ozon API-ключи
4. Готово — начинайте работу

## Облако vs Self-host

| | Облако | Self-host |
|---|---|---|
| Установка | Не нужна | Docker Compose |
| Обновления | Автоматические | Ручные |
| Бэкапы | Ежедневные | Самостоятельно |
| Данные | На наших серверах | На вашем сервере |
| Поддержка | Приоритетная | Community |
| Стоимость | От 990 ₽/мес | Бесплатно |

<Callout type="info">
Self-host версия содержит все те же функции. Облако добавляет удобство: автообновления, бэкапы и поддержку.
</Callout>
