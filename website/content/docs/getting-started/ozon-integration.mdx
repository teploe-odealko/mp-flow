## Получение API-ключей Ozon

1. Войдите в [кабинет продавца Ozon](https://seller.ozon.ru)
2. Перейдите в **Настройки → API-ключи**
3. Создайте новый ключ с правами **Admin** (нужен для полного доступа)
4. Скопируйте **Client-ID** и **API-Key**

<Callout type="warn">
Не делитесь API-ключами. Они дают полный доступ к вашему кабинету продавца.
</Callout>

## Подключение в OpenMPFlow

### Через веб-интерфейс

1. Откройте Admin UI
2. Перейдите в раздел **Каталог**
3. Нажмите кнопку **Ozon** (вверху справа)
4. Введите Client-ID и API-Key
5. Нажмите **Сохранить креды**

### Через переменные окружения

Добавьте в `.env`:

```bash
OZON_CLIENT_ID=123456
OZON_API_KEY=your-api-key-here
```

## Синхронизация данных

После подключения доступны команды синхронизации:

| Функция | Описание |
|---------|----------|
| **Импорт товаров** | Загрузка каталога товаров с Ozon в систему |
| **Синхронизация остатков** | Получение текущих остатков на складах Ozon FBO |
| **Синхронизация продаж** | Импорт продаж за период с автоматическим FIFO-расчётом COGS |
| **Синхронизация финансов** | Импорт финансовых операций Ozon (комиссии, логистика, возвраты) |

### Через AI-агента

Если вы подключили AI-агента через MCP, он может запускать синхронизацию за вас:

- `ozon_import_products` — импорт каталога
- `ozon_stocks` — текущие остатки
- `ozon_sync_sales` — синхронизация продаж
- `ozon_sync_finance` — синхронизация финансов

## Поддерживаемая модель

OpenMPFlow поддерживает **Ozon FBO** (Fulfillment by Ozon) — товары хранятся на складах Ozon, доставка силами Ozon.

<Callout type="info">
Поддержка Wildberries планируется в будущих версиях.
</Callout>
