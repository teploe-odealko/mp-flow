# Плагины

Плагины добавляют в MPFlow интеграции с маркетплейсами. Каждый плагин подключает свой маркетплейс: синхронизирует товары, остатки, продажи, финансовые операции и расширяет возможности AI-агентов.

## Доступные плагины

## Как включить плагин

Перейдите в раздел интеграции в боковом меню. Плагины, доступные для вашего аккаунта, отображаются в списке. Включите нужный плагин и следуйте инструкции по настройке.
