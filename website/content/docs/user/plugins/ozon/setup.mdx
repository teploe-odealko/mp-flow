# Подключение аккаунта Ozon

Для работы плагина нужны два ключа из личного кабинета Ozon Seller: **Client-Id** и **API Key**.

## Шаг 1. Получите ключи в Ozon Seller

1. Откройте [личный кабинет Ozon Seller](https://seller.ozon.ru/)
2. Перейдите в **Настройки** → **Seller API**
3. Нажмите **Сгенерировать ключ**
4. Выберите нужные права доступа (роль **Admin** для полного доступа)
5. Скопируйте **Client-Id** и **API Key**

<Callout type="info">
API Key показывается только один раз при создании. Сохраните его в безопасном месте.
</Callout>

## Шаг 2. Добавьте аккаунт в MPFlow

1. В MPFlow откройте раздел **Ozon** в боковом меню
2. Перейдите на вкладку **Аккаунты**
3. Нажмите **Добавить аккаунт**
4. Введите название (например, "Основной магазин"), **Client-Id** и **API Key**
5. Нажмите **Проверить** — система проверит что ключи валидны
6. Сохраните аккаунт

## Шаг 3. Запустите синхронизацию

После добавления аккаунта перейдите на вкладку **Синхронизация** и нажмите **Синхронизировать все**. Система загрузит:

- Товары и привяжет их к карточкам MPFlow
- Остатки FBO на складах Ozon
- Продажи и возвраты
- Финансовые операции (комиссии, логистика, начисления)

Первая синхронизация может занять несколько минут в зависимости от количества товаров.

## Несколько аккаунтов

Можно подключить несколько магазинов Ozon. Каждый аккаунт синхронизируется независимо. Данные разделены по аккаунтам — товары одного магазина не смешиваются с другим.
