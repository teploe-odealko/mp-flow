# Плагин Ozon

Плагин подключает ваш магазин на Ozon к MPFlow. После настройки система автоматически синхронизирует товары, остатки, продажи, возвраты и финансовые операции.

## Возможности

- **Синхронизация товаров** — карточки Ozon привязываются к карточкам MPFlow, данные обновляются автоматически
- **Остатки FBO** — актуальные остатки на складах Ozon
- **Продажи и возвраты** — FBO-отправления, статусы доставки, возвраты с причинами
- **Финансы** — комиссии, логистика, начисления по каждой продаже
- **AI-управление** — полный доступ к Ozon Seller API через AI-агента
