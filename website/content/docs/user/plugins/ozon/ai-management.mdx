# Управление магазином через AI-агента

Плагин Ozon добавляет AI-агентам прямой доступ к Ozon Seller API. Это значит, что подключённый агент (Claude Code, Cursor, или любой другой MCP-совместимый клиент) может выполнять любые операции с вашим магазином на Ozon — от просмотра товаров до обновления цен.

<Callout type="info">
Для подключения AI-агента нужен API-ключ MPFlow. Инструкция по настройке — в разделе [AI-агенты](/docs/user/ai-agents).
</Callout>

## Что доступно агенту

Плагин добавляет инструмент `ozon_api`, который является прокси ко всему Ozon Seller API — **более 400 методов** в 50+ категориях:

| Категория | Что можно делать |
|-----------|-----------------|
| Товары | Получить список товаров, описания, характеристики |
| Цены | Обновить цену, минимальную цену, зачеркнутую цену |
| Остатки | Проверить остатки FBO/FBS, обновить остатки FBS |
| Продажи | Список отправлений, статусы, детали заказов |
| Финансы | Транзакции, отчёты по начислениям, комиссии |
| Аналитика | Отчёты по продажам, воронка конверсий, трафик |
| Акции | Список акций, добавить/убрать товары из акций |
| Склады | Информация о складах, остатки по складам |

## Справочник API

Агенту также доступен встроенный справочник Ozon Seller API в виде MCP-ресурсов. Агент может запросить список всех категорий API или детали конкретной категории, чтобы найти нужный метод. Это позволяет агенту самостоятельно определить, какой эндпоинт вызвать для выполнения задачи.

## Примеры использования

Вот что можно попросить AI-агента сделать:

### Цены
> "Установи минимальную цену на воронку 1000 рублей"

Агент найдёт товар, определит нужный метод API и обновит цену.

### Аналитика
> "Покажи все товары с остатком меньше 10 штук"

> "Какие товары продаются лучше всего за последний месяц?"

### Карточки товаров
> "Покажи описание и характеристики товара «подставка для ноутбука»"

### Отправления
> "Сколько заказов было за последнюю неделю? Есть ли возвраты?"

## Безопасность

Действия AI-агента зависят от прав API-ключа Ozon, который вы указали при подключении аккаунта. Агент использует эти ключи для вызовов к Ozon API.

**Не используйте API-ключ Ozon с правами, которые вы не готовы доверить агенту.** Если вы хотите ограничить агента только просмотром данных — создайте в Ozon Seller отдельный API-ключ с правами только на чтение.

Ozon позволяет создавать ключи с разными ролями:
- **Admin** — полный доступ (чтение + запись)
- Ограниченные роли — только определённые разделы API

Выбирайте роль в соответствии с тем, что вы хотите разрешить агенту делать.
