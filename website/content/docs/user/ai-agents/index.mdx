# AI-агенты

MPFlow работает как MCP-сервер, что позволяет подключить к вашему личному кабинету любого AI-агента, поддерживающего протокол [MCP](https://modelcontextprotocol.io/) (Model Context Protocol).

## Что такое MCP

MCP — открытый протокол, который позволяет AI-агентам подключаться к внешним системам и работать с ними. Агент получает список доступных инструментов, понимает что каждый из них делает, и вызывает нужные для выполнения вашей задачи.

## Что доступно агенту

Подключённый агент получает доступ к основным функциям MPFlow:

| Инструмент | Описание |
|------------|----------|
| Каталог | Просмотр, создание, редактирование товаров |
| Склад | Остатки, поступления, продажи, списания по каждому товару |
| Закупки | Прогноз закупок, создание заказов поставщикам |
| Поставщики | Список заказов, детали, статусы |
| Продажи | Список продаж с фильтрами по статусу, каналу, дате |
| Финансы | P&L отчёт, транзакции |
| Аналитика | Юнит-экономика, стоимость остатков |

Плагины расширяют этот список. Например, [плагин Ozon](/docs/user/plugins/ozon/ai-management) добавляет прямой доступ ко всему Ozon Seller API.

## Совместимые AI-агенты

MCP поддерживается в:

- **Claude Code** — CLI-агент от Anthropic
- **Claude Desktop** — десктопное приложение
- **Cursor** — IDE с AI-ассистентом
- Любой другой клиент, поддерживающий MCP Streamable HTTP транспорт

Инструкция по подключению — в разделе [Настройка](/docs/user/ai-agents/setup).
