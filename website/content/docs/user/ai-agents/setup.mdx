# Подключение AI-агента

## Шаг 1. Создайте API-ключ

1. Откройте MPFlow и перейдите в **Настройки** (иконка шестерёнки в боковом меню)
2. В разделе **API-ключи** нажмите **Создать ключ**
3. Введите название (например, "Claude Code")
4. Скопируйте ключ — **он показывается только один раз**

## Шаг 2. Добавьте конфигурацию

### Claude Code

Создайте файл `.mcp.json` в корне проекта:

```json
{
  "mcpServers": {
    "mpflow": {
      "type": "http",
      "url": "https://admin.mp-flow.ru/mcp",
      "headers": {
        "Authorization": "Bearer ВАШ_API_КЛЮЧ"
      }
    }
  }
}
```

### Cursor

Создайте файл `.cursor/mcp.json` в корне проекта:

```json
{
  "mcpServers": {
    "mpflow": {
      "url": "https://admin.mp-flow.ru/mcp",
      "headers": {
        "Authorization": "Bearer ВАШ_API_КЛЮЧ"
      }
    }
  }
}
```

### Self-hosted

Если вы развернули MPFlow самостоятельно, замените URL на адрес вашего сервера:

```
http://ваш-сервер:3000/mcp
```

<Callout type="info">
Готовые конфигурации для копирования также доступны на странице **Настройки** в MPFlow.
</Callout>

## Шаг 3. Проверьте подключение

После добавления конфигурации перезапустите агента. Он должен обнаружить инструменты MPFlow автоматически.

Попробуйте задать вопрос:

> "Покажи список товаров в MPFlow"

Если агент вернул данные — подключение работает.

## Управление ключами

- Каждый ключ можно отозвать в любой момент в **Настройки** → **API-ключи**
- Создавайте отдельные ключи для разных агентов и проектов
- В таблице видно дату последнего использования каждого ключа
