OpenMPFlow — open source ERP-система для продавцов на маркетплейсах (Ozon FBO).

## Что такое OpenMPFlow?

OpenMPFlow управляет полным циклом продаж на маркетплейсе:

- **Каталог** — карточки товаров, обогащение данными поставщиков
- **Закупки** — заказы поставщикам, распределение общих затрат, приёмка
- **Складской учёт** — FIFO по партиям, остатки, синхронизация с Ozon
- **Аналитика** — юнит-экономика, PnL, ДДС
- **Ценообразование** — калькулятор прибыли с комиссиями Ozon, массовое обновление цен
- **Акции** — массовое управление акциями Ozon с защитой маржи
- **Логистика** — матрица перемещений SKU, поставки на Ozon
- **AI-агент** — 54 MCP инструмента для Claude, ChatGPT и других AI-клиентов

## Быстрый старт

```bash
git clone https://github.com/nickthecook/mpflow
cd mpflow
cp .env.example .env
docker compose up
```

Откройте [http://localhost:3000](http://localhost:3000) и войдите с логином `admin` / `admin`.
