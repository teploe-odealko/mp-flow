MPFlow — open source ERP-система для продавцов на маркетплейсах (Ozon FBO).

## Что умеет MPFlow

| Модуль | Что делает |
|--------|-----------|
| **Каталог** | Карточки товаров, обогащение данными поставщиков |
| **Закупки** | Заказы поставщикам, распределение затрат, приёмка |
| **Складской учёт** | FIFO по партиям, остатки, синхронизация с Ozon |
| **Аналитика** | Юнит-экономика, PnL, ДДС |
| **Ценообразование** | Калькулятор прибыли с комиссиями Ozon, массовое обновление цен |
| **Акции** | Массовое управление акциями Ozon с защитой маржи |
| **AI-агент** | 54+ MCP инструментов для Claude, ChatGPT и других клиентов |
