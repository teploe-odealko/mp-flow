# MPFlow

Открытая AI Agent First система управления маркетплейсами. Каталог, закупки, склад, продажи, финансы и интеграция с Ozon — в одном месте. Подключайте AI-агентов через MCP для управления магазином.

## Возможности

- **Каталог** — карточки товаров с артикулами, размерами, ценами, данными поставщиков
- **Закупки** — заказы поставщикам с распределением общих затрат (логистика, таможня)
- **Складской учёт** — партионный учёт остатков с точной себестоимостью
- **Продажи** — автоматическая синхронизация с Ozon (FBO-отправления, возвраты, комиссии)
- **Финансы** — транзакции, P&L, юнит-экономика по товарам
- **AI-агенты** — MCP-сервер для подключения Claude Code, Cursor, Codex и других AI-клиентов
- **Плагины** — расширяемая архитектура (Ozon, 1688.com, Фото-студия из коробки)

## Tech Stack

| Слой | Технология |
|------|-----------|
| HTTP | Hono 4 |
| ORM | MikroORM 6 + PostgreSQL 17 |
| DI | awilix |
| Auth | Logto OIDC |
| AI | MCP (Model Context Protocol) |
| Frontend | React 19 + Vite 6 + Tailwind CSS |
| Data fetching | TanStack Query 5 |
