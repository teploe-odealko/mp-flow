## Отчёты

OpenMPFlow формирует два ключевых финансовых отчёта с реальной себестоимостью по FIFO.

## PnL — Прибыли и убытки

Отчёт PnL показывает финансовый результат за период:

| Показатель | Описание |
|------------|----------|
| Выручка | Продажи на Ozon |
| Себестоимость (COGS) | Рассчитана по FIFO |
| Валовая прибыль | Выручка − COGS |
| Операционные расходы | Комиссии Ozon, логистика, возвраты |
| Чистая прибыль | Валовая прибыль − Расходы |

### Группировка

Данные можно группировать по:
- **Дням** — детальный анализ
- **Неделям** — среднесрочная динамика
- **Месяцам** — стратегический обзор

## ДДС — Движение денежных средств

Отчёт ДДС показывает реальное движение денег:

| Категория | Примеры |
|-----------|---------|
| Поступления | Продажи, возвраты средств |
| Расходы | Закупки, логистика, комиссии |
| Итого | Чистый денежный поток |

## Ручные транзакции

Для затрат, которые не синхронизируются автоматически (аренда, зарплаты, реклама), используйте ручные финансовые транзакции.

## MCP инструменты

| Инструмент | Параметры | Описание |
|------------|-----------|----------|
| `reports_pnl` | `date_from`, `date_to`, `group_by` | Отчёт прибылей и убытков |
| `reports_dds` | `date_from`, `date_to` | Движение денежных средств |
| `finance_list` | `limit`, `offset` | Список финансовых транзакций |
| `finance_create` | `amount`, `type`, `category`, `description` | Добавление транзакции |
