## Юнит-экономика по SKU

Отчёт юнит-экономики показывает прибыльность каждого товара с учётом реальной себестоимости по FIFO.

### Что включает отчёт

Для каждого SKU за выбранный период:

| Показатель | Описание |
|------------|----------|
| Выручка | Сумма продаж на Ozon |
| Себестоимость (COGS) | Рассчитана по FIFO — из реальных партий закупок |
| Комиссии Ozon | Комиссия площадки, логистика, эквайринг |
| Валовая прибыль | Выручка - COGS - комиссии |
| Маржа | Валовая прибыль / Выручка × 100% |
| ROI | Валовая прибыль / COGS × 100% |

### Почему FIFO важен для юнит-экономики

Средневзвешенная себестоимость скрывает реальную картину. Если вы закупили партию дешевле, а следующую дороже — FIFO покажет точную прибыль каждой продажи.

## Как использовать

### Через веб-интерфейс

Раздел **Отчёты → Юнит-экономика**. Выберите период и получите таблицу по всем SKU.

### Через AI-агента

```
Покажи юнит-экономику за последний месяц
```

Агент вызовет `reports_unit_economics` и покажет результаты.

## MCP инструмент

| Инструмент | Параметры | Описание |
|------------|-----------|----------|
| `reports_unit_economics` | `date_from`, `date_to` | Отчёт юнит-экономики по SKU за период |
