## Управление акциями

MPFlow позволяет массово управлять участием товаров в акциях Ozon с защитой от продаж в убыток.

## Минимальная цена с защитой маржи

Для каждого товара система рассчитывает минимальную цену акции:

```
Минимальная цена = Себестоимость FIFO + Комиссии Ozon + Целевая маржа
```

Это гарантирует, что даже при участии в акции вы не будете продавать в минус.

## Функции

### Массовое включение в акции

- Выберите акцию Ozon
- Система покажет список ваших товаров с рекомендованной минимальной ценой
- Включите участие для всех подходящих товаров одним действием

### Массовое выключение

- Отключите товары из акции, если условия не выгодны

### Автообновление таймеров

- Минимальная цена на Ozon действует ограниченное время
- MPFlow автоматически обновляет таймер, чтобы цена не сбросилась

## Как это работает

1. **Анализ** — система берёт себестоимость по FIFO и считает все комиссии
2. **Расчёт минимальной цены** — добавляет целевую маржу
3. **Установка** — отправляет минимальную цену через Ozon API
4. **Контроль** — следит за таймерами и обновляет при необходимости

<Callout type="info">
Минимальная цена — это цена, ниже которой Ozon не сможет опустить товар в акции. Она защищает вашу маржу.
</Callout>
