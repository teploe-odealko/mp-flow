## Матрица перемещений SKU

Матрица показывает полный путь каждого товара:

| Этап | Описание |
|------|----------|
| Заказано | Количество в заказах поставщикам |
| В пути | Отгружено поставщиком, ещё не на складе |
| На складе | Принято, доступно для отправки на Ozon |
| На Ozon FBO | Остатки на складах Ozon |
| Продано | Реализовано через Ozon |
| Списано | Потери, брак, расхождения |

## Поставки на Ozon

Управление поставками товаров на склады Ozon FBO.

## Контроль потерь

### Списание потерь

Если товар потерян или повреждён на складе Ozon:
- `logistics_write_off_loss` — списание с расчётом убытка по FIFO себестоимости

### Списание расхождений

При инвентаризации, если фактические остатки не совпадают с учётными:
- `logistics_write_off_discrepancy` — корректировка с указанием причины

## MCP инструменты

| Инструмент | Описание |
|------------|----------|
| `logistics_matrix` | Матрица перемещений всех SKU |
| `logistics_supplies` | Список поставок на Ozon |
| `logistics_sku_detail` | Детальная информация по SKU |
| `logistics_write_off_loss` | Списание потерь |
| `logistics_write_off_discrepancy` | Списание расхождений |
