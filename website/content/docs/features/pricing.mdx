## Калькулятор прибыли

MPFlow рассчитывает прибыль с учётом всех комиссий Ozon:

```
Цена продажи           2,490 ₽
───────────────────────────────
Себестоимость FIFO     -  850 ₽
Комиссия Ozon (15%)    -  374 ₽
Last mile              -  120 ₽
Эквайринг (1.5%)       -   37 ₽
УСН (6%)               -  149 ₽
───────────────────────────────
Прибыль                   960 ₽
ROI                      113%
Маржа                   38.6%
```

## Два режима расчёта

### Forward: Цена → Маржа

Задайте цену продажи — система покажет прибыль, маржу и ROI с учётом всех расходов.

### Reverse: Маржа → Цена

Задайте желаемый ROI или маржу — система рассчитает минимальную цену продажи.

## Учитываемые комиссии

| Комиссия | Описание |
|----------|----------|
| Комиссия Ozon | По категории товара (5-25%) |
| Last mile | Доставка покупателю |
| Эквайринг | Обработка платежа (1.5%) |
| Возвраты | Процент возвратов |
| УСН | Налог (6% или настраиваемый) |

Ставка УСН настраивается в **Настройки → УСН**.

## Массовое обновление цен

Можно обновить цены на все товары одним действием:
1. Установите целевой ROI (например, 40%)
2. Система рассчитает цену для каждого SKU
3. Проверьте и примените изменения

## MCP инструмент

| Инструмент | Параметры | Описание |
|------------|-----------|----------|
| `pricing_calculate` | `card_id`, `price` или `target_roi` | Расчёт цены или прибыли для товара |
