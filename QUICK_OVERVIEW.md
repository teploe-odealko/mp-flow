# MPFlow — Краткий обзор для презентации

## Что это?
**AI Agent First ERP** для продавцов на Ozon. Управляйте товарами, закупками, складом, финансами и подключайте AI-агентов через MCP.

## 5 главных фич

### 1️⃣ **Себестоимость по партиям** (как в 1С)
- Каждая партия имеет свою себестоимость
- Распределение затрат: логистика, таможня, упаковка
- FIFO/LIFO при продажах
- **Ключевое отличие от других сервисов** — честная экономика

### 2️⃣ **Автопрогноз закупок с визуализацией** ⭐
- Анализирует историю продаж
- Рассчитывает когда закончатся остатки
- Красиво показывает на timeline bar
- Один клик → заказ поставщику
- **Самая впечатляющая фича для видео**

### 3️⃣ **P&L + Юнит-экономика**
- Выручка - Себестоимость - Комиссии = Прибыль
- ROI по каждому товару
- Разбивка комиссий (FBO, маркетинг, логистика)
- Видно какие товары убыточные

### 4️⃣ **Синхронизация с Ozon**
- Товары ↔ Ozon (двусторонняя)
- Остатки в реальном времени
- Продажи автоматические
- Финансы уже учтены

### 5️⃣ **Управление через AI**
- Подключи Claude Code / Cursor / Codex через MCP
- Агент видит 54+ инструмента
- Естественный язык: "Поднять цены на товары < 10% маржи на 15%"
- AI сам вызовет API и сделает

```

```

## Workflows (самые важные)

| Workflow | Что делает |
|----------|-----------|
| **Receive Order** | Поступление товара → себестоимость + распределение затрат |
| **Create Sale** | Продажа → списание со склада (FIFO) → финансовая транзакция |
| **Write Off** | Списание товара (брак) → убыток |
| **Sync Ozon** | Автосинхронизация всех данных с Ozon |
| **Analyze** | P&L, юнит-экономика, стоимость остатков |

## Как использовать

### Облако (быстро)
- **admin.mp-flow.ru** — 14 дней бесплатно, потом 300 ₽/мес

### Self-hosted (бесплатно)
```bash
curl -O https://raw.github.../docker-compose.yml
cat > .env << EOF
DB_PASSWORD=ваш_пароль
COOKIE_SECRET=$(openssl rand -base64 32)
EOF
docker compose up -d
# http://localhost:3000
```

### Для разработчиков
```bash
git clone https://github.com/teploe-odealko/mp-flow.git
cd admin
npm install
# .env с DATABASE_URL и COOKIE_SECRET
npm run dev  # localhost:5173 (клиент) + 3000 (API)
```
