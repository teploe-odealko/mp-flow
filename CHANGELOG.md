# Changelog

Все значимые изменения в проекте документируются в этом файле.

## [Unreleased]

### Добавлено

- **MCP-сервер** — Streamable HTTP транспорт на `/mcp` для подключения AI-агентов (Claude Code, Cursor, Codex и др.)
- **API-ключи** — генерация и управление ключами для MCP и REST API
- **OpenAPI Playground** — интерактивная документация API на `/api/docs`
- **Плагин 1688.com** — поиск поставщиков и импорт товаров с 1688.com
- **Плагин Фото-студия** — генерация фото товаров через AI (OpenRouter)
- **Биллинг** — подписка, токены, тарифы для облачной версии
- **Сайт документации** — docs.mp-flow.ru на Fumadocs (Next.js)
- **Лэндинг** — редизайн mp-flow.ru с анимированным hero
- **S3 хранилище** — загрузка файлов через MinIO (S3-совместимое)
- **Настройки** — страница настроек с управлением API-ключами и интеграциями

## [0.1.0] — 2026-03-02

Первый публичный релиз.

### Добавлено

- **Каталог** — карточки товаров с SKU, размерами, ценами, данными поставщиков
- **Закупки** — заказы поставщикам с распределением общих затрат (логистика, таможня)
- **Складской учёт** — партионный учёт с точной себестоимостью каждой единицы
- **Юнит-экономика** — PnL по товару с реальной себестоимостью
- **Продажи** — синхронизация с Ozon (FBO-отправления, возвраты, комиссии)
- **Финансы** — транзакции, ДДС, отчёты
- **Аналитика** — сводные отчёты по продажам и складу
- **Плагины** — convention-based auto-discovery архитектура
- **Ozon плагин** — синхронизация товаров, остатков, продаж, возвратов, финансовых транзакций
- **Auth** — Logto OIDC авторизация
- **Self-hosting** — Docker Compose деплой
